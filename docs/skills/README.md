# iGED Implementation Skills

Implementation guidelines derived from project plans. Use these when working on the corresponding features.

| Skill | Description |
|-------|-------------|
| [saved-filters-toolbar](saved-filters-toolbar/SKILL.md) | Saved filter presets, adaptive picker, overlay expansion |
| [vue-ensemble-route](vue-ensemble-route/SKILL.md) | Dashboard Vue d'ensemble route, tabs, KPI cards, expandable tables |
| [adapt-table-structure](adapt-table-structure/SKILL.md) | Single-header table with aligned expandable detail rows |
| [iged-dashboard-figma](iged-dashboard-figma/SKILL.md) | Full dashboard implementation from Figma (ITCSS, BEMIT, Plectrum) |

## Usage

- **Cursor**: Skills are mirrored in `.cursor/skills/` for auto-discovery. Edit `docs/skills/` as source of truth and sync to `.cursor/skills/` when updating.
- **Other editors** (VS Code, JetBrains, etc.): Reference these markdown files when implementing the listed features.
- **AI assistants**: Point to the relevant SKILL.md when asking for help on a specific area.
